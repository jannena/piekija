# Käyttäjäin hallinta
## Uuden käyttäjän luonti
Uuden käyttäjän voi lisätä järjestelmään henkilökuntanäytön Käyttäjät-välilehdellä valitsemalla *Lisää uusi käyttäjä*. Tämän jälkeen selain tekee uudelleenohjauksen käyttäjän sivulle, jossa näkyvät kyseisen käyttäjän lainat ja jossa voi muokata käyttäjän tietoja.

## Käyttäjän hakeminen
Käyttäjän voi hakea henkilökuntanäytön *Käyttäjät*-välilehdellä käyttäjän nimen tai viivakoodin perusteella. Käyttäjän viivakoodin voi selvittää myös hakemalla *Lainaa ja palauta* -näkymässä jonkin teoksen, joka on lainattu käyttäjälle. Käyttäjän viivakoodi näkyy kohdassa *Tila*.

## Uuden henkilökuntakäyttäjän lisääminen
Ohjeet tähän löytyvät ohjeiden kohdasta *Kehittäjälle* --> *Asennus*.
