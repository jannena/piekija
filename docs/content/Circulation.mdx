# Lainaaminen
Nimikkeen voi lainata käyttäjälle henkilökuntanäytön *Lainaa ja palauta*-välilehdellä. Sivulla on kaksi kenttää. Ylempään valitaan käyttäjän viivakoodi (joka löytyy kirjastokortista) ja alempaan nimikkeen viivakoodi (joka löytyy nimikkeestä).

![Kuva 1](/docs/img/circulation-1.png)

Valitun nimikkeen voi lainata käyttäjälle painamalla *Lainaa käyttäjälle...*

# Palauttaminen tai tilan tarkastaminen
Nimikkeen palauttamiseksi tai tilan tarkastamiseksi riittää, että hakee *Lainaa ja palauta* -välilehdeltä vain nimikkeen tiedot nimikkeen viivakoodilla.

![Kuva 1](/docs/img/circulation-2.png)

Nyt nimikkeen voi palauttaa painamalla *Palauta*.
