# Etusivu-uutiset
Henkilökuntanäytön *Etusivu-uutiset*-välilehdellä voi hallita koko järjestelmän etusivulla näkyviä uutisluonteisia ilmoituksia. Etusivulla käyttäjille näkyy aina vain **viisi** uusinta etusivu-uutista, vaikka joskus henkilökuntakäyttäjät saattavatkin nähdä etusivulla useamman etusivu-uutisen.
