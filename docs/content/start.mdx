# Tervetuloa PieKiJan käyttäjäksi!
PieKiJa on pienten koulu- ja kotikirjaston käyttöön tarkoitettu pieni kirjastojärjestelmä (siitä muuten nimikin PieKiJa). Näissä ohjeissa opastetaan lyhyesti järjestelmän käyttöön. Mukana on käyttöliittymän ohjeiden lisäksi myös kehittäjille tarkoitettu REST-rajapinnan dokumentaatio.

Mukavia herkiä PieKiJan parissa.
